# AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes, calculates ATS scores, matches resumes with job descriptions, and provides AI-powered suggestions for improvement.

## Features

- Upload PDF/DOCX resumes
- Resume Parsing
- ATS Score
- Job Description Matching
- Skill Gap Analysis
- AI Suggestions
- Dashboard
- Download Reports

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express

Database
- Firebase

AI
- OpenAI API
